# 📚 ปพ.5 — ระบบบันทึกการพัฒนาคุณภาพผู้เรียน

ระบบสมุด ปพ.5 สำหรับครูระดับประถมศึกษา พัฒนาเป็น Web App ใช้งานได้ทันทีในเบราว์เซอร์ ไม่ต้องติดตั้งโปรแกรม

---

## ✨ ฟีเจอร์หลัก

### 📋 จัดการรายวิชา
- สร้างได้หลายวิชา ครูสอนกี่วิชาก็รองรับ
- ตั้งสัดส่วนคะแนนเก็บ/ปลายภาคได้อิสระ (70/30, 60/40, 80/20)
- รายชื่อนักเรียนชุดเดียวใช้ร่วมกันทุกวิชา

### 📅 บันทึกเวลาเรียน
- ตารางแสดงชื่อนักเรียนทุกคนพร้อมกัน เช่นเดียวกับสมุด ปพ.5 จริง
- คลิกปุ่มสลับสถานะ: **ว่าง → ✓ มา → ม ขาด → ส สาย → ล ลา**
- ค่าเริ่มต้นเป็นช่องว่าง ไม่สมมุติว่ามาเรียน จนกว่าครูจะเช็ค
- **ตั้งวันที่อัตโนมัติ** — ระบุวันเริ่มเรียน + วันในสัปดาห์ที่เรียน ระบบสร้างวันที่ทุกคาบให้ทั้งภาคเรียน
- **แก้ไขวันที่ได้ทุกช่อง** — รองรับวันชดเชย/สลับวัน
- **ช่องหมายเหตุรายคาบ** — ระบุ "วันหยุด", "สอบ", "เรียนชดเชย" ได้
- กรอกเร็วด้วยปุ่ม "✓ มาทั้งหมดทุกคน" หรือ "✓ คาบที่ X มาทุกคน"

### 📋 คะแนนตัวชี้วัด
- เพิ่มตัวชี้วัดได้ไม่จำกัด ตามคำอธิบายรายวิชา
- กรอกคะแนนเต็มของแต่ละตัวชี้วัดได้เลย ไม่แบ่งช่วง
- แสดงผ่าน/ไม่ผ่านรายตัวชี้วัดทันที

### ✏ คะแนนสอบ
- สอบปลายภาคครั้งเดียว ทั้งภาค 1 และภาค 2
- คำนวณรวมกับคะแนนเก็บอัตโนมัติ

### 🌟 การประเมินด้านอื่น
- **คุณลักษณะอันพึงประสงค์** 8 ด้าน (ระดับ 0-3)
- **การอ่าน คิดวิเคราะห์และเขียน** (ระดับ 0-3)

### 📊 สรุปผลการเรียน
- **ภาคเรียน**: แสดงคะแนนเก็บ + สอบ แต่ยังไม่ออกระดับผล
- **ทั้งปี**: นำคะแนนภาค 1 + ภาค 2 มาเฉลี่ย → ระดับผล 0–4 ตามเกณฑ์ประถมศึกษา
- แสดงจำนวนตัวชี้วัดที่ผ่าน/ไม่ผ่านรายคน
- แผนภูมิการกระจายระดับผลการเรียน

### 🖨 พิมพ์รายงาน
เลือกพิมพ์ได้ทีละรายการหรือทั้งหมดพร้อมกัน:
- บันทึกการเข้าเรียน (พร้อมวันที่และหมายเหตุ)
- ผลการประเมินตัวชี้วัด
- คะแนนสอบปลายภาค
- คุณลักษณะอันพึงประสงค์
- การอ่าน คิดวิเคราะห์และเขียน
- สรุปผลการเรียนทั้งปี

## 📐 โครงสร้างการคำนวณผลการเรียน

```
ภาคเรียนที่ 1 (100 คะแนน)
├── คะแนนเก็บ (ตามสัดส่วน เช่น 70)
│   └── ผลรวมตัวชี้วัดทุกตัว
└── สอบปลายภาค 1 ครั้ง (เช่น 30)

ภาคเรียนที่ 2 (100 คะแนน)
├── คะแนนเก็บ (ตามสัดส่วน)
└── สอบปลายภาค 1 ครั้ง

ผลการเรียนทั้งปี
= (ภาค 1 + ภาค 2) ÷ 2
→ แปลงเป็นระดับผล 0–4
```

### เกณฑ์ระดับผลการเรียน

| ระดับ | ความหมาย | ช่วงคะแนน |
|---|---|---|
| 4 | ดีเยี่ยม | 80–100 |
| 3.5 | ดีมาก | 75–79 |
| 3 | ดี | 70–74 |
| 2.5 | ค่อนข้างดี | 65–69 |
| 2 | น่าพอใจ | 60–64 |
| 1.5 | พอใช้ | 55–59 |
| 1 | ผ่านเกณฑ์ | 50–54 |
| 0 | ต่ำกว่าเกณฑ์ | 0–49 |

> ⚠️ นักเรียนต้องผ่านเกณฑ์เวลาเรียน ≥ 80% ทั้ง 2 ภาค จึงจะได้รับระดับผลการเรียน
